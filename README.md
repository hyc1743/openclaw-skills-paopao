# OpenClaw Skills - Paopao

Personal collection of OpenClaw skills for AI model provider management and automation.

## Skills

### model-provider-manager

Manage AI model providers and models in OpenClaw configuration.

**Features:**
- Add new model providers and models
- Update existing configurations
- Remove providers
- Troubleshooting guide

**Usage:** Download `model-provider-manager.skill` and install it in your OpenClaw skills directory.

## Installation

1. Download the `.skill` file
2. Place it in your OpenClaw skills directory
3. Restart OpenClaw
